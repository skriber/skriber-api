# Skriber

> Attention: Currently skriber is not production ready. 
> Use with caution!

[Skriber](https://skriber.co/) is a simple yet performant websocket server. Manage the realtime
needs of your apps with ease. Main features are:

- 🔁 Pub/Sub - Model
- 🔐 Private Channels
- Multi-Tenancy
